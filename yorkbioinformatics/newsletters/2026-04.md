      - title: Introduction to Scientific Computing
        link: https://www.york.ac.uk/biology/technology-facility/data-science/courses/
        image: /yorkbioinformatics/assets/images/intro-to-sci-comp-hero-image.png
        description: |
          On this course, you'll become familiar with the core concepts of Scientific Computing, a skill necessary for handling the sorts of big data generated in your scientific research.<br/>
          You'll learn the basics of using Bash to navigate and control your computer from the terminal, through to more advanced techniques like accessing other computers (e.g. High Performance Computing / HPC clusters) remotely.<br/>
          We'll also teach you how to use version control systems like Git and remote code repositories like GitHub to manage the code that you write.
        date: 20–21 May 2026
        location: Biology Department, The University of York
        
      - title: Introduction to Python Programming
        link: https://www.york.ac.uk/biology/technology-facility/data-science/courses/
        image: /yorkbioinformatics/assets/images/intro-to-python-course-hero-image.png
        description: |
          Learn the basics of the world's most popular programming language — Python — and then apply your newly-learned skills to begin to analyse scientific data. Suitable for beginners, or those who have dabbled a little in another language.<br/>
          We'll cover the fundamentals of programming, then move on to more advanced concepts like functions and classes, before moving on to reading in, processing, and plotting real data.<br/>
          By the end, you'll be proficient enough in Python to apply it to your everyday work and research.
        date: 27–28 May 2026
        location: Biology Department, The University of York
