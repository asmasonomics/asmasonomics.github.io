Enter the newsletters directory, copy the template and edit to create the month's newsletter.<br/>
The new document should be called <code>YYYY-MM.md</code>

Once complete, commit and then visit the newsletter website. For example:<br/> [https://asmasonomics.github.io/yorkbioinformatics/newsletters/april-2026.html](https://asmasonomics.github.io/yorkbioinformatics/newsletters/april-2026.html)
Then select all, copy and paste into a gmail compose window before sending round.
